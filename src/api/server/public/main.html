Stackoverflow-ish

What is Stackoverflow-ish

Well, Stackoverflow-ish is just like Stackoverflow, but the terminal edition.

How

It is written in Go, Python and Node.JS; The Node.JS part, is a server that stores all the Q/A things.
The Python part of this project, is the API client. It asks the server for data.

Example

For example

$ python3 api.py -rd pass123
